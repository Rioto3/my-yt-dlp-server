# YouTube Audio Extractor

YouTubeから簡単に高品質な音声を抽出するツールキット。Firefox拡張機能とバックエンドAPIを組み合わせた、シンプルで使いやすいソリューション。

## 主な機能

- 🎵 YouTubeから高品質な音声（MP3）を抽出
- 🖱️ ワンクリックダウンロード
- 🌐 Firefox拡張機能対応
- 🚀 高速かつ簡単な操作

## 使用方法

1. Dockerコンテナを起動
2. YouTubeページを開く
3. 動画の下に表示される「MP3を保存」ボタンをクリック
4. 高品質MP3を自動ダウンロード

## 技術スペック

- **音声形式**: MP3
- **音質**: 320 kbps (固定)
- **対応動画**: YouTubeの公開動画
